# cropland-fragmentation-grain-yield
Data &amp; Code for "Decoding the Restrictive Coupling Mechanisms Between Cropland Fragmentation and Grain Yield at Optimal Spatial Scales"
## 03_Landscape_Metrics/
Contains 9 landscape metrics calculated by Fragstats 4.2 (moving window, granularity=180m, amplitude=1080m) for 6 years (2000, 2005, 2010, 2014, 2018, 2022).
Metrics: AI, AREA_MN, CLUMPY, COHESION, DIVISION, LSI, NP, PARA_MN, PD
